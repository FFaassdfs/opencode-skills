# Task Scheduler Checklist

## 生成 Windows 计划任务前必须检查

1. 是否有稳定入口文件（推荐 `.bat`）
2. `.bat` 是否切换到了正确目录：`cd /d "..."`
3. `.bat` 是否去掉了 `pause`
4. 是否需要 `chcp 65001 >nul`
5. 脚本依赖的环境变量在计划任务账户下是否存在
6. Python / OpenCode 路径是否可发现
7. 计划任务命令是否引用了完整路径
8. 是否有日志或清晰报错

## 推荐模式

不要把复杂命令直接塞进 `/tr`。

推荐：

- `schtasks` 调 `.bat`
- `.bat` 调 Python
- Python 执行业务逻辑

## 常见坑

- 手工运行正常，计划任务失败：通常是 PATH 或工作目录不同
- 中文路径导致脚本异常：需正确引用路径
- GUI 程序被计划任务启动但无输出：优先使用 CLI 或服务 API
- 计划任务窗口卡住：通常是 `.bat` 内残留 `pause`
